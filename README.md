# TaskFlow — Professional Task Manager

A modern, feature-rich task management app built with React + Vite + Tailwind CSS v4.

## Features
- 📋 **List View** — grouped by date, drag-to-reorder, subtasks
- ⊞ **Board View** — Kanban-style with drag & drop between columns  
- 📅 **Calendar View** — monthly view with task dots
- 🕐 **History** — full activity timeline (created, completed, edited, deleted)
- 📊 **Analytics** — completion rate, category breakdown, activity heatmap
- ⏱️ **Pomodoro Timer** — focus timer per task
- 🎤 **Voice Input** — speech-to-text task entry
- 📁 **Attachments** — images & files per task
- 📝 **Markdown Notes** — rich text descriptions
- 🔁 **Repeating Tasks** — daily/weekly/monthly
- ☀️/🌙 **Dark/Light Theme**
- 📤 **Export** — CSV & PDF download

## Setup

```bash
npm install
npm run dev
```

## Tech Stack
- React 18
- Vite 5
- Tailwind CSS v4 (with @apply)
- Syne + DM Sans + JetBrains Mono fonts
- LocalStorage persistence
